synchronized关键字作用于方法和代码块，对应为**同步方法**和**同步代码块**，其中同步方法只需在方法中加入关键字，同步代码块需要设置具体给哪个对象加锁

**同步方法例**：

```java
public synchronized void funcName(){
	do something
}
```

**同步代码块**

```java
synchronized(Object object){	
	do something;
}
```

**6种用法**

- 同步代码块

  - 加锁对象为本地变量
  - 类静态变量
  - 共享变量
  - 类对象

- 同步方法

  - 修饰普通方法
  - 修饰类静态方法
